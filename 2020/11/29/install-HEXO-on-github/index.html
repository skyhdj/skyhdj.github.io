install HEXO on github

1、 什么是HEXO
2、 什么是GitHub
Hexo是一款用于搭建博客的框架，用户不需要在写自己的代码利用别人写好的来搭建既方便又单，像这样的还有wordpress等等开源框架

GitHub是一个代码托管平台2019年被以72亿美元收购，拥有给世界大概10亿用户的以及个网站，方便与你的代码或者是你公司代码项目的托管，值得每个程序员学习和使用

来说说hexo的搭建

首先我们需要下载安装node.js https://nodejs.org/zh-cn/（国内的可能会更快一点）
主要看一下node的版本和npm的版本 node -v npm -v

我们主要是用npm来安装的但是呢有个问题就是这个源在国外用起来会比较慢，
我推荐使用国内的源、使用npm install -g cnpm –registry=https://registry.npm.taobao.org
(使用淘宝的)

直接安装：cnpm install -g hexo-cli
Hexo -v 验证一下是否安装

开始搭建博客：
在你的本地创建的文件夹，我的是D:\blog
通过cmd控制台更方便，进入到blog目录
初始化你的博客 hexo init 执行代码后会去GitHub远程克隆文件

这时候\blog下就会有这些文件
#####到这里安装就结束了
通过hexo start 来启动你的博客也可以使用简写 hexo s(鄙人比较懒)

哎嘿！这时候它会给你个地址http://localhost:4000 这里它默认使用的是4000端口(注意一下)

你的可能只是有下面hello world这个页面，我的是安装了主题的（下面会讲别急），
继续#####
我们来写一篇新的文章 使用hexo n(new的缩写) hexo n “我的第一篇文章”，你可以利用自带的vim 或者vscode来编辑 我用的是:code 我的第一篇文章.md（注意生成的是markdown语法）这里挖个坑以后讲。

建好之后呢返回到你的\blog文件下从新生成 清理一下文件 hexo clear ——在生成
Hexo g生成呢就是看看你刚才操作了啥 给你整理整理，在重启你的服务 hexo s

劲爆的来了！！！！！
通过GitHub来获取免费的公网体验（听懂的掌声），登陆到你的GitHub

这里注意（必须是你的GitHub用户名格式是这样的例：skyhdj.github.io）别问问就是官方要求。

我们需要把本地的项目部署到GitHub端，这里我们需要一个git 小插件、利用cnpm来安装 cnpm install –save hexo-deployer-git

在配置本地的__config.yml文件##

找到#deployment 添加 type、repo、branch,repo是你的GitHub仓库地址。

部署到云端 hexo d部署

这里需要注意 当出现错误时看一下 这两个文件 git config -global user.name ‘yourname’和
git config -global user.email ‘youremail’写上你自己的就可以了。

最后呢它会提示你输入你的GitHub账号和密码 在刷新你的GitHub就可以了很方便，
